# Pomodoro Timer

## Live Demo
https://pomodoro-timer-x28b.vercel.app

## How to Run

### Option 1 — Using npx serve (recommended)
npx serve .
Then open http://localhost:3000 in your browser.

### Option 2 — Open directly
Just open index.html directly in any browser. No build step needed.

## Features
- Configurable focus and break durations
- Visual countdown ring with color-coded states
- Audible chime when a session ends
- Daily session history saved across page reloads
- Keyboard shortcuts: Space to start/pause, R to reset
- Fully responsive — works on mobile and desktop

## Stack
Vanilla HTML, CSS, JavaScript — no frameworks, no dependencies.
